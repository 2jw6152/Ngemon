# 게임 이름: 느겜몬

## 1. 핵심 내용
- 포켓몬 배틀을 핵심으로 한 온라인 게임.
- CPU 배틀을 먼저 개발 후 온라인으로 확장 예정.
- 리액트, PixiJS, TypeScript, Vite 기반으로 제작함.
- 추후 데이터베이스로 Supabase 연결 예정
- 다크 모드 기반에 네온 포인트, 글래스모피즘(유리 질감) 패널 사용한 현대적인 디자인의 UI로 구성할 것.
- 화면 좌측 바에 메뉴 보드가 있음. 구성은 대시보드 / 배틀 / 상점 / 파티 / 설정정

## 2. 핵심 규칙
- 본가 포켓몬 게임의 배틀 시스템을 그대로 가져옴.
- PP 시스템: 기술 사용 횟수(PP)는 배틀 시작 시 풀 충전되며, 배틀이 끝나면 다시 초기화
- 플레이어에게 노출되는 타입(속성), 기술명, UI 문구는 반드시 한국어로 표시할 것

## 3. 게임 경제 및 흐름
- 시작 포켓몬: 계정 생성 시 모든 포켓몬 리스트 중 랜덤으로 3마리 지급. 기술은 모든 포켓몬이 4개씩 가지며 처음에도 중복 없이 가능한 기술 중 4개를 랜덤 습득.
- 상점: 포인트(pt)를 화폐 단위로 함. 상단바로 포켓몬 / 기술 / 아이템 탭을 변경 가능
  - 포켓몬
    - 랜덤 포켓몬 (300pt): 모든 포켓몬 중 랜덤으로 1마리 포켓몬을 뽑음. 이때 총 종족값이 높을수록 나올 확률을 적게 함.
    - 슈퍼 팩 (1500pt): 모든 포켓몬 중 랜덤으로 5마리 포켓몬을 순서대로 뽑음. 이 경우 종족값에 의한 확률 변동이 없으며 모든 포켓몬이 균등한 확률로 뽑힘. 또 확정적으로 종족값 총합이 500pt 이상인 포켓몬이 마지막에 1마리 나옴
  - 기술
    - 랜덤 기술 변경 (50pt): 선택한 포켓몬의 선택한 기술을 랜덤으로 변경함. (기존 기술과 중복 불가. 배울 수 있는 기술 중에서)
    - 선택 기술 변경 (300pt): 선택한 포켓몬의 선택한 기술을 원하는 기술로 변경함. (기존 기술과 중복 불가. 배울 수 있는 기술 중에서)
  - 아이템
    - 아이템 뽑기 (100pt): 추후 구현. 현재는 탭만 구성
- 배틀에서 승리 시 100pt 얻음.

## 4. 배틀 엔진
- 턴 우선순위 및 데미지
  - 우선도가 높은 기술이 먼저 나감
  - 우선도가 같다면 스피드가 높은 쪽이 선공.
  - 스피드까지 같다면 랜덤으로 선공
  - 데미지 계산: 물리/특수 분류에 따라 공격/방어 스탯을 적용하고 제공된 타입 상성표를 통해 최종 데미지 산출 (battle formula 참고)

## 5. 화면 구성
- 대시보드
- 배틀: 초기 단계에는 CPU 배틀 버튼만 삽입
- 상점: ##3의 상점 참고
- 파티: 상단에 현재 파티, 하단에 보유 포켓몬 목록. 포켓몬들은 카드 형태로 띄울 것. 현재 파티는 카드를 드래그&드랍해 파티 순서를 변경 가능
- 설정: 초기 단계에는 로그아웃 버튼만 삽입
